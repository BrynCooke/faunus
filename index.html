FAUNUS

Graph Analytics Engine

Download this project as a .zip file

Faunus is a Hadoop-based graph processing framework for analyzing massive-scale graphs. A breadth-first version of the graph traversal language Gremlin operates on graphs stored in the distributed graph database Titan, in any Rexster-fronted graph database, or in HDFS via various text and binary formats.

In addition, Faunus provides the following features:

  • Scalable, distributed global graph processing.
  • Support for various input and output formats:
    • Aurelius Titan
      • Apache Cassandra
      • Apache HBase
    • TinkerPop Rexster
    • GraphSON Text Files
    • RDF Text Files
    • Hadoop Binary Files
  • Native integration with the TinkerPop graph stack:
    • Gremlin graph query language
    • Blueprints standard graph API
  • Open source with the liberal Apache 2 license.

Download Faunus or clone from GitHub. Read the Faunus documentation and join the mailing list.

For commercial Faunus support, please contact the graph experts at Aurelius.
