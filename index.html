Faunus

A Library of Graph-Based Hadoop Tools

Faunus is a library for processing graphs using Hadoop's MapReduce framework. Faunus provides operators that implement a property graph algebra that can be used to process massive scale graphs.

  • Support for various data source
    • Titan 0.1
    • Hadoop File System 1.0
  • Natively implements the Blueprints graph API
  • Open source with the liberal Apache 2 license

Faunus is maintained by Aurelius
