Faunus

A Library of Hadoop-Based Graph Tools

Faunus is a framework for processing property graphs using Hadoop. A vertex-centric, adjacency list data structure is used whereby every record is a single vertex along with its incident edges. This vertex-centric data structure is then processed using map-steps (e.g. filters, mutations) and reduce-steps (e.g. traversals). MapReduce operations are provided that implement a property graph algebra for doing graph derivations and basic graph statistics. Faunus can be extended with new operations written using MapReduce and Blueprints.

  • Support for various data sources/sinks
    • Titan 0.1 distributed graph database
    • Rexster 2.x fronted graph databases that support vertex iteration
    • GraphSON text format stored in HDFS
  • Support for various MapReduce-based graph algorithms
    • Numerous graph derivation/transformation steps
    • Numerous global graph statistics
  • Natively implements the Blueprints graph API
  • Open source with the liberal Apache 2 license
